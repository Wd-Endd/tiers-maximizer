# tiers-maximizer
Minecraft Vibrant Visuals tools.

# Needed
* [NodeJS](https://nodejs.org)
* [NPM](https://npmjs.com) 
-   Clone this Repository and open terminal this cloned directory.
-   Overwrite tiers.bin by latest file from Minecraft Assets.
-   Folow these command.

> For NPM:
```bash
npm install
node ./app.js
```
> For Yarn:
```bash
yarn install
node ./app.js
```
